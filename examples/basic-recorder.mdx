# Basic Recorder Example

This example shows a complete, minimal implementation of the React Voice Visualizer for recording audio.

## Complete Code

<CodeGroup>

```jsx App.jsx
import React from 'react';
import { useVoiceVisualizer, VoiceVisualizer } from 'react-voice-visualizer';

function App() {
  // Initialize the hook with no parameters for basic usage
  const recorderControls = useVoiceVisualizer();

  return (
    <div style={{ padding: '20px' }}>
      <h1>Voice Recorder</h1>
      <VoiceVisualizer 
        controls={recorderControls}
        height={200}
        width="100%"
      />
    </div>
  );
}

export default App;
```

## How It Works

### 1. Import the Components

First, import both the hook and the visualizer component:

```jsx
import { useVoiceVisualizer, VoiceVisualizer } from 'react-voice-visualizer';
```

### 2. Initialize the Hook

The `useVoiceVisualizer` hook manages all the recording state and provides controls:

```jsx
const recorderControls = useVoiceVisualizer();
```

This returns a `Controls` object with properties and methods for:
- Starting/stopping recording
- Playing back recorded audio
- Accessing audio data and metadata

### 3. Render the Visualizer

Pass the controls to the `VoiceVisualizer` component:

```jsx
<VoiceVisualizer 
  controls={recorderControls}
  height={200}
  width="100%"
/>
```

## Available Controls

The `recorderControls` object provides access to:

- **State**: `isRecordingInProgress`, `isPausedRecording`, `isCleared`
- **Audio Data**: `recordedBlob`, `audioData`, `bufferFromRecordedBlob`
- **Time Info**: `recordingTime`, `duration`, `currentAudioTime`
- **Methods**: `startRecording()`, `stopRecording()`, `togglePauseResume()`, `clearCanvas()`
- **Formatted Time**: `formattedRecordingTime`, `formattedDuration`, `formattedRecordedAudioCurrentTime`

<Note>
The component automatically requests microphone permissions when the user clicks the record button.
</Note>
